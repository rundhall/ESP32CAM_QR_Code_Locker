SETUP

1. Copy your generated codes.csv into:
   data/codes.csv

2. Copy the qr_codes directory generated by Python next to:
   public/
   data/
   qr_codes/

3. Edit public/reset.php

   $resetCode = 'RESET-XXXXXX';

4. Create data/state.json:

   {
      "current_index": 0
   }

5. Open public/index.php

Behavior:
- Correct answer => next unused QR.
- Each QR is distributed only once.
- State stored server-side in state.json.
- flock() prevents duplicate allocation during concurrent requests.
